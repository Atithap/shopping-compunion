## การตั้งค่าสำหรับผู้ใช้งานใหม่

เนื่องจากโปรเจกต์นี้ใช้ระบบความปลอดภัยด้วยไฟล์ `.env` ผู้ใช้งานใหม่ต้องทำตามขั้นตอนดังนี้:

1. **เตรียม API Key:** ไปสร้าง Key ของตัวเองที่ [Google AI Studio](https://aistudio.google.com/)
2. **สร้างไฟล์ .env:** - คัดลอกไฟล์ `.env.example` แล้วเปลี่ยนชื่อเป็น `.env`
   - นำ API Key ที่ได้มาวางแทนที่คำว่า `your_api_key_here`
3. **ติดตั้งและรัน:**
   ```bash
   npm install
   npm run build
